# Cryptography

## Overview

This section documents my study of fundamental cryptographic concepts and password cracking techniques used in cybersecurity. The goal is to understand how cryptographic mechanisms protect information, how password hashes are stored, and how security professionals perform password recovery and auditing in authorized environments.
